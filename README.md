# ✨ Experience Highlights

### 🌎 Interactive 3D Globe

A custom-built Three.js powered Earth visualization synchronized with user scrolling, creating a futuristic and engaging hero experience.

### ⚡ GSAP Motion System

Premium animations powered by GSAP including:

* ScrollTrigger-driven storytelling
* Staggered reveals
* Smooth section transitions
* Horizontal scrolling experiences
* Advanced parallax effects

### 🧠 Neural Network Visuals

Custom HTML5 Canvas neural network simulations that bring life and intelligence to the website background.

### 🔊 Immersive Sound Design

Integrated ambient audio system with:

* Browser-safe playback handling
* User-controlled sound toggles
* Smooth fade transitions

### 🚀 Performance Focused

Built for speed and scalability:

* Lazy-loaded components
* Optimized asset delivery
* Hardware-accelerated animations
* Efficient render pipelines

### 📱 Fully Responsive

Fluid layouts and adaptive typography ensure a seamless experience across:

* Desktop
* Laptop
* Tablet
* Mobile

# ⚙️ Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/shieldx.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
